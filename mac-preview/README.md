# CleanCheck Preview

Double-click `Launch CleanCheck Preview.command` to start a tiny local server on your Mac.

Then open the iPhone URL shown in Terminal in Safari on your iPhone while both devices are on the same Wi-Fi.

Any changes to the files in this folder reload automatically in the browser.

This is a live preview tool for web-style app files. It is not a way to install a signed native iPhone app directly onto the phone.
